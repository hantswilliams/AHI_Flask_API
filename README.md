AHI_Flask_API

- Each version increases in complexity (Build_1 through Build_X): 
    - 1: Simple `GET` request
    - 2: `GET` request -  ALL data from python dictionary 
    - 3: `GET` request - specific data using filters
    - 4: `POST` request 
    - 5: Password protected routes 

- To run locally: 
    - `python Build_1.py` 
    - `python Build_2.py`
    - etc....
